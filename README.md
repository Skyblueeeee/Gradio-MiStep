快速开始
1.拉取代码

`git clone https://github.com/Skyblueeeee/Gradio-MiStep.git`

2.新建虚拟环境

`conda create -n tool_env==3.8`

3.下载安装依赖

`pip install -r requirements.txt -i 国内镜像源`

4.bat文件路径修改

`PYTHON_PATH="虚拟环境路径"`
